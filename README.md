### Installation

1. Clone the repo

   ```bash
   $ git clone https://github.com/xartd0/Discord-RPC-Yandex.Music.git
   $ cd Discord-RPC-Yandex.Music
   $ pip install pypresence
   $ pip install PyQt5
   $ pip install PyQtWebEngine
   $ python3 core.py
   ```
