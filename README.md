### :running: Запуск локально

Клонируем репозиторий

```bash
  git clone https://github.com/xartd0/Discord-RPC-Yandex.Music.git
```

Заходим в его папку

```bash
  cd Discord-RPC-Yandex.Music
```

Ставим зависимости

```bash
  pip3 install -r requirements.txt
```

Запуск

```bash
  python3 core.py
```
